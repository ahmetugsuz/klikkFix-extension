Privacy Policy for KlikkFix

Last updated: 04.04.2025

1. What data does KlikkFix collect?

KlikkFix collects no personal data. All text processing happens locally in your browser, and no text is stored or sent to external servers without your consent.

2. Use of External Services

KlikkFix uses the OpenAI API for text improvement. The text sent to the API is processed temporarily to generate improvements but is not stored by the extension.

3. Permissions

For KlikkFix to function properly, it requires the following permissions:

  • contextMenus: Used to add KlikkFix to the right-click menu.
  • scripting: Used to manipulate text directly on web pages.
  • activeTab: Used to fetch the text from the active tab when you activate the extension.
  • Host permissions: Used to connect to KlikkFix's server for text improvements.

4. How can you delete your data?

Since KlikkFix does not store any user data, there is nothing to delete.

5. Contact

If you have any questions regarding privacy, you can contact us at: ahmetugsuz123@gmail.com
