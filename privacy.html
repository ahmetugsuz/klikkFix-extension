Personvernerklæring for KlikkFix

Sist oppdatert: [DATO]

1. Hvilke data samler KlikkFix?

KlikkFix samler ingen personlige data. Tekstbehandlingen skjer lokalt i nettleseren din, og ingen tekst lagres eller sendes til eksterne servere uten ditt samtykke.

2. Bruk av eksterne tjenester

KlikkFix bruker OpenAI API for tekstforbedring. Teksten som sendes til API-et behandles midlertidig for å generere forbedringer, men lagres ikke av utvidelsen.

3. Tillatelser

For å fungere krever KlikkFix tilgang til:

  • contextMenus: Brukes for å legge til KlikkFix i høyreklikkmenyen.
  • scripting: Brukes for å manipulere tekst direkte på nettsider.
  • activeTab: Brukes for å hente teksten på den aktive fanen når du aktiverer utvidelsen.
  • Vertstillatelser: Brukes for å koble til KlikkFix sin server for tekstforbedringer.

4. Hvordan kan du slette dine data?

Siden KlikkFix ikke lagrer noen brukerdata, er det ikke nødvendig å slette noe.
